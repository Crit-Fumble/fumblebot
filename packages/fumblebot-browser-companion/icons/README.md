# Extension Icons

Add the following icon files to this directory:

- `icon-16.png` - 16x16 pixels (toolbar, favicon)
- `icon-48.png` - 48x48 pixels (extension management page)
- `icon-128.png` - 128x128 pixels (Chrome Web Store, installation)

You can use the Crit-Fumble logo or create custom icons for the extension.

## Recommended Approach

Copy or resize from existing project assets:
- `packages/fumblebot-browser-companion/node_modules/@crit-fumble/react/dist/img/crit-coin.png`
- Or use the favicon from the web app
